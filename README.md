# 🌸 Lovely Friend — 可愛的朋友

一個 Fabric mod，為 Minecraft 添加了一種全新的友好生物——**朋友（Friend）**。

朋友會自主生活、戰鬥、探索、與你互動，像一個真正的夥伴一樣陪伴你冒險。

## 🧑‍🤝‍🧑 朋友基本介紹

**朋友（Friend）** 是一種智能路徑尋找的友好生物，具有以下特點：

- ❤️ **生命值**：20（10 顆心）
- 🏃 **移動速度**：0.5（與玩家走路速度相近）
- ⚔️ **基礎攻擊力**：3 點
- 🛡️ **基礎護甲**：2 點
- 🎒 **背包**：100 格空間
- 🌾 **飢餓系統**：會隨著時間消耗飽食度，需要進食
- 💕 **可以繁殖**：兩個朋友可以生出小朋友

### 基本能力

| 能力 | 說明 |
|:----|:------|
| 🎒 **拾取物品** | 自動撿起地上的物品放入背包 |
| ⚔️ **戰鬥** | 能使用劍、弓、弩、三叉戟戰鬥 |
| 🌾 **進食** | 餓了會自動從背包找食物吃 |
| ⛏️ **使用工具** | 會自動裝備最佳的工具 |
| 🚰 **游泳** | 不會淹死（FloatGoal） |
| 🔗 **可以被拴繩** | 玩家可以用拴繩拉著朋友走 |
| 🐷 **可以騎乘** | 玩家可以騎在朋友身上 |

## 🎁 取得朋友

### 自然生成
朋友只會在**村莊**中自然生成。

### 繁殖
給兩隻朋友餵食，牠們會產生後代。繁殖後有 5 分鐘冷卻時間。

## 🧠 朋友的行為 AI（Goal 系統）

朋友最厲害的地方在於牠們的**自主決策系統**。朋友會根據當下的環境、狀態和需求，自動選擇要做什麼事。所有行為按照 **優先級（Priority）** 排列，數字越小優先級越高。

### Priority 0 — 核心本能（生存第一！）

| Goal | 說明 | 觸發條件 |
|:----|:-----|:---------|
| 🌊 **FloatGoal** | 在水中浮起來避免淹死 | 在水中時 |
| 🍔 **EatFoodGoal** | 餓了自動從背包拿出食物吃 | 飽食度 < 20、背包有食物 |
| 🔧 **EquipBestToolGoal** | 手上沒工具時自動從背包裝備最佳工具 | 手上是空的 |
| 🌽 **FindFoodGoal** | 肚子餓又沒食物時去尋找食物來源（成熟的作物或動物） | 飽食度 < 20、背包無食物 |

### Priority 1 — 危險防禦（火、爆炸）

| Goal | 說明 | 觸發條件 |
|:----|:-----|:---------|
| 🔥 **ExtinguishFireGoal** | 身上著火時找最近的水源滅火 | 身上著火 |
| 💥 **FleeDangerGoal** | 逃離火焰、TNT、即將爆炸的苦力怕 | 附近有危險源頭 |

### Priority 2 — 戰鬥行為

| Goal | 說明 | 觸發條件 |
|:----|:-----|:---------|
| 🏹 **RangedAttackGoal** | 有弓/弩和箭矢時遠程射擊敵人 | 有遠程武器和彈藥 |
| ⚔️ **MeleeAttackGoal** | 近戰攻擊敵人 | 有近戰目標 |
| 🎯 **MoveTowardsTargetGoal** | 朝目標敵人移動 | 有目標敵人 |

**目標選擇**（TargetSelector）：
- 反擊任何傷害朋友的生物
- 主動攻擊怪物（苦力怕除外）
- 主動攻擊動物（作為食物來源）

### Priority 3 — 環境互動與工具製作

| Goal | 說明 | 觸發條件 |
|:----|:-----|:---------|
| 📦 **CollectItemOnGroundGoal** | 撿起地上掉落的物品 | 附近有物品、空閒時 |
| 🐷 **RideFriendGoal** | 騎到其他朋友身上 | 附近有可騎乘的朋友 |
| ⛏️ **DiggingEscapeGoal** | 🔥 **卡住時挖開阻礙！** | **重點行為，見下方詳解** |
| 🧱 **PlaceBlockToClimbGoal** | 前方有高處時放置方塊爬上去 | 導航目標在上方且卡住 |
| 🪵 **CraftWoodenToolsGoal** | 檢查背包木頭量，足夠就自動合成木鎬、木斧、木劍 | 缺少工具時自動計算所需木材 |

#### ⛏️ DiggingEscapeGoal（挖掘逃脫）— 詳細說明

這是朋友最關鍵的自主行為之一，讓朋友在卡住時**自己挖路**。

**觸發流程：**
1. 朋友發現在導航但超過 **1.5 秒（30 ticks）** 沒移動
2. 判斷可能被方塊卡住
3. 找出擋路的方塊開始挖掘

**挖掘速度 — 完全參考 Minecraft 玩家挖掘機制：**

| 工具材質 | 對應方塊速度 | 挖石頭時間 | 挖泥土時間 |
|:-------:|:----------:|:---------:|:---------:|
| 🖐️ 空手 | 1.0（懲罰 5x） | **11.25 秒** | 0.9 秒 |
| 🪵 木鎬 | 2.0 | **1.125 秒** | — |
| 🪨 石鎬 | 4.0 | **0.56 秒** | — |
| 🔩 鐵鎬 | 6.0 | **0.375 秒** | — |
| 💎 鑽石鎬 | 8.0 | **0.28 秒** | — |
| 🔥 獄髓鎬 | 9.0 | **0.25 秒** | — |
| ✨ 金鎬 | 12.0 | **0.187 秒** | — |

> ⚠️ 用錯工具（如斧頭挖石頭）或空手挖需要工具才能破壞的方塊，會有 **5 倍懲罰**。

**智慧選工具：** `equipBestToolForBlock()` 會自動在背包中找出對目標方塊破壞速度最快的工具來裝備。

**挖掘偏好順序：**
1. 🆕 **優先挖頭頂上方** — 掉進洞穴或坑洞時先挖天花板
2. 挖路徑前方的方塊
3. 挖路徑側邊的方塊

**徒手挖掘：** 就算完全沒有工具，朋友也會用空手慢慢挖（只是非常慢）。

### Priority 3.5 — 自主行為（覓食、種植、砍樹）

| Goal | 說明 | 觸發條件 |
|:----|:-----|:---------|
| 🐄 **HuntLivestockGoal** | 肚子餓時獵殺附近的動物（豬、牛、羊、雞） | 飽食度不足、附近有動物 |
| 🌱 **PlantCropGoal** | 有種子時在耕地種植作物 | 空閒時 |
| 🌲 **ChopTreeGoal** | 🆕 **砍木頭！砍光附近所有樹為止** | 附近有原木 |

#### 🌲 ChopTreeGoal（砍木頭）— 詳細說明

朋友一旦開始砍樹，就會**把附近所有能看到的原木都砍完才停手**！

**砍伐行為：**
- 開始後會連續砍樹，砍完一根立刻找下一根
- 會沿著樹幹從下往上砍，完整剝光整棵樹
- 砍完整棵樹後在附近繼續找下一棵
- **直到半徑 14 格內完全沒有原木才停下來**

**砍樹速度 — 使用真實玩家挖掘公式（同 DiggingEscapeGoal）：**

| 工具材質 | 對原木速度 | 砍一根原木時間 |
|:-------:|:----------:|:-------------:|
| 🖐️ 空手 | 1.0（懲罰 5x） | **11.25 秒** |
| 🪓 木斧 | 2.0 | **2.25 秒** |
| 🪨 石斧 | 4.0 | **1.125 秒** |
| 🔩 鐵斧 | 6.0 | **0.75 秒** |
| 💎 鑽石斧 | 8.0 | **0.56 秒** |
| 🔥 獄髓斧 | 9.0 | **0.5 秒** |
| ✨ 金斧 | 12.0 | **0.375 秒** |

> 💡 沒斧頭時朋友會空手砍（很慢），但只要有木頭就會做木斧！
> 💡 砍完的木頭會自動在 `CraftWoodenToolsGoal` 中合成成工具。

### Priority 4 — 環境互動（採礦、釣魚、分享）

| Goal | 說明 | 觸發條件 |
|:----|:-----|:---------|
| ⛏️ **BreakBlockGoal** | 🆕 **自主挖礦！** | Y < 60、附近有礦物 |
| 🎣 **FishingGoal** | 有釣竿時在附近水域釣魚 | 有釣竿、附近有水 |
| 🗡️ **ShareWeaponGoal** | 送武器給空手的玩家 | 附近有沒武器的玩家 |
| 🛡️ **ShareArmorGoal** | 送護甲給缺少裝備的玩家 | 附近有缺護甲的玩家 |
| 🍞 **ShareFoodGoal** | 分享食物給飢餓的玩家 | 附近有飽食度 < 18 的玩家 |
| 🔗 **LeashMobGoal** | 用拴繩套住動物牽著走 | 背包有拴繩、附近有無主的動物 |

#### ⛏️ BreakBlockGoal（挖礦）— 詳細說明

朋友會在地下（Y < 60）尋找有價值的礦物來挖掘。

**可辨識的礦物：**
- 煤礦、銅礦、鐵礦、金礦、紅石礦、青金石礦、鑽石礦、綠寶石礦
- 地獄金礦、地獄石英礦、遠古遺骸

**挖礦工具機制：**
- 🆕 **沒鎬子也能挖！** 就算手上沒鎬子，朋友還是會嘗試挖礦——只是空手挖石頭要 **15 秒**
- 挖得很慢 → 會暫時冷卻 → 冷卻期間自然會去 **砍木頭 → 做木鎬 → 回來挖更快**
- 背包中有鎬子會自動裝備最好的

### Priority 5 — 自主生活（探索、思考人生、繁殖）

| Goal | 說明 | 觸發條件 |
|:----|:-----|:---------|
| 💕 **BreedFriendGoal** | 找附近的朋友繁殖 | 空閒時、無繁殖冷卻 |
| 🚶 **RandomStrollGoal** | 隨機走動 | 空閒時、被拴繩時減少 |
| 🏘️ **StrollThroughVillageGoal** | 在村莊中散步 | 在村莊中 |
| 🌍 **ExploreGoal** | 探索新的生態域 | 空閒時（低機率） |
| 🤔 **ContemplateLifeGoal** | 🆕 **思考人生！決定遠行！** | **重點行為，見下方詳解** |

#### 🤔 ContemplateLifeGoal（思考人生）— 詳細說明

這是最有趣的行為之一！朋友偶爾會停下來「思考人生」，然後決定一個**長期的目標**並堅持朝目標前進。

**目標選擇優先序：**
1. 如果已經有一個目標且還沒抵達 → **直接繼續前進**（即使被打斷過）
2. 搜尋半徑 64 格內是否有**村莊** → 朝村莊前進
3. 沒有村莊 → 往隨機方向 **長途跋涉 100~300 格**

**目標持久化機制：**
- `targetPos` 一旦設定，會永久保存
- 即使被其他行為打斷（戰鬥、吃東西、挖礦），`targetPos` **不會消失**
- 打斷結束後，`canUse()` 檢查到還有目標 → **立即恢復前進**！不需要冷卻！
- 只有在**真正抵達目標半徑 5 格內** 時，才會清除目標，開始下一次思考冷卻

**恢復前進的場景舉例：**
```
🤔 決定前往村莊（200 格外）
   ↓
   → 路上遇到殭屍 → 🗡️ 戰鬥
   → 打完了 → 🤔 繼續朝村莊前進！（不需等待）
   ↓
   → 掉進坑洞 → ⛏️ DiggingEscapeGoal 挖天花板
   → 爬出來 → 🤔 繼續朝村莊前進！
   ↓
   → 肚子餓 → 🍔 EatFoodGoal 吃東西
   → 吃飽了 → 🤔 繼續朝村莊前進！
   ↓
   → 空手挖礦好慢 → 🌲 ChopTreeGoal 砍木頭
   → 做出木鎬 → 🤔 繼續朝村莊前進！
   ↓
📍 抵達村莊附近！🎉
```

## 🆕 行為變化摘要（最新版本）

相比於原始版本，以下是最近新增或修改的行為：

| 變更 | 說明 |
|:----|:-----|

| 🎯 **目標持久化** | 思考後的目標不會因被打斷而消失，會持續前進直到抵達 |
| 🪓 **沒鎬也挖礦** | `BreakBlockGoal` 沒鎬子也能挖，只是很慢 |
| 🌲 **砍樹不停歇** | `ChopTreeGoal` 一旦開始砍樹，會砍光附近所有原木才停 |
| 🚶 **跟隨玩家** | `FollowPlayerGoal` 新增，會主動跟隨附近玩家 |

## 💡 小提示

1. **給朋友工具**：丟一把鐵鎬給朋友，他挖礦速度會快很多
2. **拴繩帶著走**：用拴繩拉著朋友移動到你想讓牠探索的區域
3. **朋友會分享**：有多餘的食物、武器、護甲時會送給玩家
4. **注意朋友肚子餓**：飽食度太低朋友會受傷，確保附近有食物來源
